## 1. 产品定位

### 这是什么

面向**普通中文日语学习者**的中文词典 overlay。不是专业大辞典，不是学术数据库。

### 目标用户

- 日语学习者（EJU / N2-N3 水平为主）
- 生活、学校、考试、工作场景中的常见日语
- 需要快速理解词义、区分义项的中文母语用户

### 优先级

现代常用日语 > EJU/N2-N3 范围 > 生活/学校/考试/工作常见用法

### 不是以下内容

- 专业大辞典（不追求覆盖所有义项）
- 佛教辞典（佛教专门义项应隐藏或加标注）
- 古语辞典（古语义项默认隐藏）
- 法律辞典（法律细分义项默认隐藏）
- 医学专业辞典（医学制度/组织/职位细分默认隐藏）
- 方言辞典（方言义项默认隐藏）

## 2. shouldDisplay 规则

### 定义

`shouldDisplay` 表示是否**默认展示给普通学习者**。

- `shouldDisplay=true`：默认在查词结果中显示
- `shouldDisplay=false`：不默认显示（但义项仍然存在于完整数据中）

### 核心原则

| 原则 | 说明 |
|------|------|
| 不能因翻译正确就默认展示 | 翻译正确 ≠ 应该展示给普通学习者 |
| 不能因属于某领域就一律隐藏 | 医学/法律/教育领域的基础词汇仍应展示 |
| 不能因不属于某领域就一律展示 | 罕见义项、古语、专业细分必须隐藏 |
| `shouldDisplay=false` ≠ 义项不存在 | 只是默认折叠，不等于删除 |

### 应默认展示的例子

| 条目 | 原因 |
|------|------|
| 医療 | 极常用现代词 |
| 医学 | EJU/N2 范围常见词 |
| 医学部 | 升学/学校语境高频 |
| 医薬品 | 日常生活中高频出现 |
| 違法 | 虽属法律领域，但是极常用现代词 |
| 先生（敬称义项） | 学习者必知 |
| 勉強になる（经验/教训义项） | 学习者高频表达 |
| お大事に | 日常祝福语 |

### 应默认隐藏或保守处理的例子

| 条目 | 原因 |
|------|------|
| 阿弥陀（佛教专名义项） | 宗教专名，普通学习者价值低 |
| お前（古语：御前） | 古语义项 |
| 何れ（旧感叹词） | 罕见过时用法 |
| 安価（论坛术语义项） | 网络小众术语 |
| 案件（法律诉讼义项） | 法律细分义项 |
| 医局/医長（特定职位） | 医学制度/组织/职位细分 |
| 哀れむ（古语：欣赏美） | 古语义项 |
| ちび（前缀用法） | 过于小众 |

### shouldDisplay 判断流程

```
这个词义是否是现代日常日语中常用且对普通学习者有价值的？
  ├─ YES → shouldDisplay=true
  └─ NO  → 是否属于以下类型？
            ├─ 佛教专门义 → shouldDisplay=false
            ├─ 古语/历史用语 → shouldDisplay=false
            ├─ 法律细分 → shouldDisplay=false
            ├─ 医学制度/组织/职位细分 → shouldDisplay=false
            ├─ 罕见感叹词/前缀 → shouldDisplay=false
            └─ 不确定 → confidence=low + needs_human_review
```

## 3. 固定寒暄语规则

日语中有大量固定寒暄表达，翻译时容易机械处理。以下为已验证的正确做法：

### 高频固定表达速查表

| 日语 | 正确 shortGloss | 常用中文释义 | 错误写法 | usageNote |
|------|----------------|-------------|---------|-----------|
| お邪魔します | 打扰了 | 打扰了；不好意思打扰了 | ~~失陪了~~ | 进入别人家、办公室或房间时的礼貌用语。 |
| お疲れ様 | 先走了 | 辛苦了；先走了；回见 | ~~晚安~~ | 用于工作、学校等场合告别时。也可表慰劳。 |
| お願いいたします | 请您 | 请您；拜托您了 | ~~只写"拜托"~~ | 比「お願いします」更礼貌。 |
| お大事に | 请保重 | 请保重；早日康复 | — | 对病人、受伤者或身体不适者的祝福语。 |
| お世話になる | 承蒙照顾 | 承蒙照顾；受到关照 | — | 用于对给予自己帮助/照顾的人表示感谢。 |
| お先に | 先走一步 | 先走一步；先告辞 | — | 用于先行离开时的招呼语。 |
| 畏まりました | 明白了（敬语） | 明白了；好的 | ~~遵命~~ | 敬语，用于服务业或正式场合回应对方。 |
| お元気で | 保重 | 多保重；一路平安 | — | 用于较长期的分别时的祝福。 |

### 原则

1. shortGloss 优先现代自然中文口语，不要古语化
2. usageNote 必须写清楚使用场景和礼貌程度
3. 不要仅凭日语音读/汉字直接翻译
4. 不确定时用中文概括，不编造日语例句

## 4. 中文自然度规则

### family terms（家庭称呼）

| 日语/场景 | ❌ 不要用 | ✅ 应该用 |
|-----------|---------|---------|
| お母さん（wife 义项/丈夫称呼妻子） | 妻子 | 孩子他妈；妻子 |
| お父さん（husband 义项/妻子称呼丈夫） | 丈夫 | 孩子他爸；丈夫 |
| 御主人（对方丈夫敬称） | 您丈夫 | 您先生；您丈夫 |

### 称谓/称呼自然化

| 日语 | ❌ 避免 | ✅ 推荐 |
|------|--------|--------|
| 女中 | 女佣 | 家政人员；帮佣；保姆 |
| お嬢さん（年轻女性义项） | 只写"小姐" | 年轻女性；小姐 |
| お祖母さん（老年女性义项） | 老大娘 | 老奶奶；老妇人 |

### 常用词优先自然中文口语

| 日语 | ❌ 生硬/过时 | ✅ 自然中文 |
|------|-----------|-----------|
| 畏まりました | 遵命 | 明白了（敬语）；好的 |
| お土産 | 特产 | 伴手礼；土特产；旅游纪念品 |
| 医薬品 | — | 药品 |
| ぎりぎり | 极限 | 勉强刚好；刚刚好 |
| お話中 | 通话中 | 正在通话；占线 |

### 原则

1. **shortGloss 优先现代自然口语**，不要选过时/生硬/古语词汇
2. **家庭称呼语境中**，wife/husband 不要直译"妻子/丈夫"
3. **避免带有时代/阶级/歧视感**的旧用语
4. **方言词替换为标准中文**（老大娘 → 老奶奶）
5. **多义词选最常见义**作为 shortGloss（小姐优先→年轻女性）

## 5. usageNote 规则

### usageNote 应该写什么

1. **使用场景**：什么时候用、对谁用
2. **礼貌程度**：敬语/口语/粗鲁/中性
3. **文体**：书面/口语/古语/专业用语
4. **防止误解的关键信息**
5. **常见的搭配或短语**

### usageNote 不应该写什么

1. 编造的不自然日语例句
2. 对学习者无用的民俗/历史细节
3. 过于冗长的背景说明
4. 英语/罗马字

### 好的写法例子

| 条目 | usageNote |
|------|-----------|
| お邪魔します | 进入别人家、办公室或房间时的礼貌用语。 |
| お疲れ様 sense 2 | 用于工作、学校等场合告别时。 |
| お願いいたします | 比「お願いします」更礼貌。 |
| 勉強になる | 常见于「勉強になる」，表示从中学到东西。 |
| 畏まりました | 敬语，用于服务业或正式场合回应对方。 |
| お話中 | 多用于电话语境，表示对方正在通话或线路占线。 |

### 原则

- 不确定时用**中文概括**，不编造日语例句
- 如果不需要就**留空**，不要为填满而写
- usageNote 是**给中文母语学习者**看的，不是给日语 native 看的

## 6. shortGloss 规则

### 定义

`shortGloss` 是义项的**紧凑中文标签**，用于快速浏览和列表展示。

### 规则

1. 一个义项一个 `shortGloss`，最多 6-8 个中文字
2. 优先选择**最常见、最自然**的中文对应词
3. 如果义项有多个近义译法，选最贴近现代日常用法的
4. 不要机械翻译英文 gloss
5. 优先中文口语自然搭配而非字典式解释

### 优先级

```
现代自然中文口语词 > 中性标准中文词 > 专业/古语/生硬词
```

## 7. QA 规则

### 自动化 QA checker 应检查

1. **shouldDisplay 领域词二次判断**：医学/法律/教育标签词是否真的应该隐藏？
2. **固定寒暄语专项**：お邪魔/お疲れ/畏まりました 等是否正确？
3. **家庭称呼专项**：wife/husband 义项是否直译为"丈夫/妻子"？
4. **旧/歧视/方言用语扫描**：女佣/老大娘/遵命 等
5. **shortGloss 自然度**：是否选择了过时或生硬的词汇？
6. **usageNote 空值检查**：需要 usageNote 的固定表达是否有说明？
7. **宗教/古语/法律/医学细分** shouldDisplay 检查
