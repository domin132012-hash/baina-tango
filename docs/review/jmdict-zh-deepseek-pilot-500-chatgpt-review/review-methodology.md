# Review Methodology

Use this standard when reviewing the Top 500 candidate.

1. Prefer ordinary Chinese-speaking Japanese learners.
2. Prefer EJU / N2-N3 learner usefulness.
3. Display high-frequency modern Japanese senses first.
4. Specialized terms, archaic senses, dialect, and rare senses should usually not be default-visible.
5. shouldDisplay means default display eligibility, not whether the sense exists.
6. A correct translation is not enough reason to default-display a sense.
7. If a usageNote is uncertain, do not invent Japanese examples.
8. Chinese glosses should read like dictionary glosses, not machine-translated sentences.
9. shortGloss should be short enough for default UI display.
10. Flags should be conservative and evidence-based.
